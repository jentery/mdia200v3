According to WebAIM's 2023 report,

Take a minute to write alt text for an image.

WebAIM stands for "Web Accessibility In Mind."

More about Alt Text

Alternative (Alt) Text is:

1. Read aloud to users by reader software,

2. Indexed by search engines,

3. Meant to convey the "why" of the image
as it relates to the content of its page, and

4. Intended to display on the page
if the image fails to load.

This language is from "Write Helpful Alt Text to Describe Images" by Digital Accessibility at Harvard.

Tips for Alt Text

Add alt text to non-decorative images.

Keep it short and descriptive.

Be literal (denotation): write what's seen.

Don't include "image of" or "photo of."

Consider the context and how it relates
to other content on the page.

This language is from "Write Helpful Alt Text to Describe Images" by Digital Accessibility at Harvard.
